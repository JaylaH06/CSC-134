## 🧠 One Key Idea Before We Start

The hardest part for beginners isn't syntax — it's **knowing when to make a new function**. My rule of thumb:

> *"If you'd explain it as a separate step to a friend, it's probably its own function."*

Steps in a trivia game:
1. Show the question
2. Get the player's answer
3. Check if it's right
4. Update the score
5. Show final results

Each of those → its own function. That's exactly how we'll build it.
